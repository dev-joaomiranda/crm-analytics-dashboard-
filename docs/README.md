# CRM Analytics Dashboard

Projeto de análise de dados focado em CRM e vendas.

## Objetivo

Criar um conjunto de dados, modelo relacional e dashboards em Power BI para responder perguntas de negócio como:

- Quais campanhas trazem mais receita e melhor ROI?
- Como está o funil de conversão (visitantes → leads → oportunidades → vendas)?
- Qual o comportamento de compra dos clientes (ticket médio, recorrência, churn)?
- Quais segmentos de clientes são mais valiosos?

## Stack utilizada

- **Banco de dados / dados**: CSV + scripts SQL (PostgreSQL ou MySQL)
- **BI**: Power BI (modelagem de dados, DAX, dashboards)
- **Linguagem de apoio (opcional)**: Python para geração de dados fictícios

## Estrutura do repositório

data/               # arquivos CSV com dados fictícios

sql/                # scripts de criação de tabelas e KPIs

pbix/               # arquivo Power BI Desktop (.pbix)

docs/               # documentação e imagens
